# cloudstaff

Sample Playwright automation framework using TypeScript.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the sample test:
   ```bash
   npm test
   ```

3. Run the test in headed mode:
   ```bash
   npm run test:headed
   ```

4. Open the report after a test run:
   ```bash
   npm run test:report
   ```

## Project structure

- `playwright.config.ts` - Playwright test configuration
- `tsconfig.json` - TypeScript compiler settings
- `src/pages` - Page object classes
- `src/tests` - Example Playwright tests

## Notes

This framework uses the Page Object Model to keep selectors and page actions organized.
